### Опис завдання

Ваше завдання полягає в тому, щоб розробити цей компонент ціноутворення та зробити його максимально наближеним до дизайну.

Ви можете використовувати будь-які інструменти, які вам подобаються, щоб допомогти вам виконати завдання. Тож, якщо у вас є щось, що ви хотіли б потренуватися, сміливо спробуйте.

Ваші користувачі повинні мати можливість:

Переглядати оптимальний макет інтерфейсу залежно від розміру екрана свого пристрою
Переглядати стани наведення курсора та фокусування для всіх інтерактивних елементів на сторінці

### Використані технології

- HTML5 для розмітки
- CSS3 стилізація
- Flexbox
- Grid

## Висновки

Даний компонент складається з трьох частин, які в основному включають лише текст. Одна колонка ззалучає використання кнопки, тож загалом практика для мене особисто була саме в побудові Grid компоненту.
